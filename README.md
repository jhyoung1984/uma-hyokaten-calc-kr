# 우마무스메 평가점 계산기 (한국어)

우마무스메 프리티 더비의 육성 평가점 계산기입니다.

## 원본 출처

- **원본 사이트**: [umakonga-t.github.io/hyokatenCalc](https://umakonga-t.github.io/hyokatenCalc/)
- **원본 저자**: [umakonga-t](https://github.com/umakonga-t)

이 프로젝트는 원본 일본어 계산기를 한국어로 번역한 버전입니다.

## 변경 사항

- UI 텍스트 전체 한국어 번역
- 스킬명 한국어 번역 (나무위키 기반)
- 캐릭터명 한국어 번역
- 오프라인 사용을 위한 데이터 내장 (`data.js`)

## 사용 방법

`index.html`을 브라우저에서 열면 바로 사용할 수 있습니다.
